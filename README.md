# Portico Exchange-frontend

````````````````````````
git clone https://github.com/PorticoExchange/PorticoExchangefrontend.git

````````
cd boltz-frontend && npm i

````````````````
make required changes as per your environment to .env file

````````
start the app


````````
npm run start
